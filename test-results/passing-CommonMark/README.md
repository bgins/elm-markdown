This markdown:


```markdown
	foo	baz		bim

```

Gives this correct output:


```html
<pre><code>foo	baz		bim
</code></pre>

```

### [Example 10](https://spec.commonmark.org/0.29/#example-10)

This markdown:


```markdown
#	Foo

```

Gives this correct output:


```html
<h1>Foo</h1>

```
